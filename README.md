# 🌍 Currency Converter

A simple and modern currency converter built with **React (Vite)** and **FastAPI**. It allows users to convert between NGN (Nigerian Naira) and other global currencies using live exchange rates from ExchangeRate-API.

## 🚀 Features

- Convert from **any currency to another**
- Real-time rates via [ExchangeRate-API](https://www.exchangerate-api.com/)
- Swap currencies with one click
- Displays the **actual exchange rate** used
- Fast frontend using **React + Vite**
- Lightweight backend using **FastAPI**

## 📦 Tech Stack

- **Frontend**: React, Vite, CSS
- **Backend**: FastAPI, Python
- **API Provider**: ExchangeRate-API
